Voici notre blog afin de vous faire suivre l'actualité de notre projet, disponible sur GitHub : reims-1018-java-le-livre.

Sprint 0

Lancement du projet

Nous allons vous présenter le projet "Le livre dont vous êtes le sorcier/ la sorcière".
Notre groupe est composé de Justine Dubois, Gaëtan Evrard, Nicolas Luzurier et Lucas Mouraud.
Le principe du projet est de vous mettre dans la peau d'un futur sorcier arrivant à Poudlard. Vous aurez à faire des choix qui permettront de vous créer votre propre histoire.

Création du storyboard

Nous avons défini le principe de base du scénario et les pages types.

Elaboration du trello

Nous avons définis certaines tâches avec leur importance et leur durée estimée en utilisant les méthodes agiles SCRUM. https://trello.com/b/qbwanNY9/livre-des-dragonneaux

Conception du message d'accueil

Nous avons décider d'accueillir nos utilisateurs par une brève phrase de bienvenue suivie de la demande du pseudonyme afin de favoriser leur immersion dans l'aventure. Pour la présentation de fin de sprint, nous avons rajouté une commande qui demande le nom de l'utilisateur. Ce nom pourra être utilisé par la suite dans l'histoire.

Sprint 1

Lecture du fichier json contenant les pages du livre

Pour le deuxième sprint, notre mission était de lier le fichier contenant nos pages à notre projet. Nous avons donc créé un tableau pour stocker toutes les informations dont on aura besoin par la suite : numéro de page, contenu, etc.

Démarrage du jeu

Après avoir demandé le pseudonyme de l'utilisateur, nous avons mis en place un système permettant de démarrer l'aventure en entrant "go" dans le terminal. Pour le moment cette commande renvoie un message "bravo", mais par la suite affichera la première page du livre.

Sprint 2

Affichage de la première page

Lors du sprint 2, nous avons réussi à lier le message "go" entré par l'utilisateur à l'appel de la page 1 du livre qui était stockée dans notre tableau de pages.
